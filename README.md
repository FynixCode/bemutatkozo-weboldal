# Bemutatkozó Weboldal

Egy egyszerű, reszponzív **személyes bemutatkozó oldal**, amely bemutatja a szolgáltatásokat, árakat és elérhetőséget. Az oldal három csomagot kínál: **Alap**, **Standard**, **Bővített**.

Ez a projekt HTML és CSS használatával készült, könnyen testreszabható saját igényeid szerint.

---

## Technológiák

- **HTML5**
- **CSS3**
- **Bootstrap** Reszponzív design (mobil és desktop kompatibilis)

---

## Fő funkciók

- Szolgáltatások bemutatása
- Árlista és csomagok (Alap / Standard / Bővített)
- Elérhetőségi információk (email)
- Egyszerű, letisztult dizájn
